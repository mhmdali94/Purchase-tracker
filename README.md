# 🧾 متتبّع أسعار المشتريات

تطبيق ويب بسيط لتسجيل مشترياتك من الموردين المختلفين، ثم البحث عن أي صنف
لمعرفة سجل أسعاره الكامل عبر كل الموردين ومن يبيعه أرخص.

- كل الأسعار بالجنيه المصري (ج.م)، مع عرض المعادل بالدولار للاطلاع فقط.
- واجهة عربية بالكامل من اليمين لليسار، تعمل على الكمبيوتر والموبايل.
- تقنيات: PHP + MySQL/MariaDB، بدون أي أدوات بناء (يعمل على XAMPP مباشرةً).

---

## 🚀 خطوات التثبيت (خطوة بخطوة)

### 1) ثبّت XAMPP
حمّل وثبّت **XAMPP** من الموقع الرسمي: <https://www.apachefriends.org>
ثم افتح **XAMPP Control Panel** وشغّل خدمتَي:

- **Apache**
- **MySQL**

### 2) انسخ ملفات التطبيق
انسخ مجلد المشروع بالكامل إلى داخل مجلد `htdocs` الخاص بـ XAMPP، وسمِّه `app`:

```
C:\xampp\htdocs\app\
```

بحيث يصبح المسار مثلاً: `C:\xampp\htdocs\app\index.php`

### 3) استورد قاعدة البيانات
1. افتح المتصفح على: <http://localhost/phpmyadmin>
2. من الأعلى اضغط تبويب **Import (استيراد)**.
3. اضغط **Choose File** واختر الملف `database.sql` الموجود داخل مجلد التطبيق.
4. اضغط **Go / تنفيذ** في الأسفل.

سيتم إنشاء قاعدة البيانات `purchase_tracker` وكل الجداول والمستخدم الافتراضي
تلقائياً بدون أي خطوات إضافية.

### 4) افتح التطبيق وسجّل الدخول
افتح المتصفح على:

```
http://localhost/app
```

ثم سجّل الدخول بالحساب الافتراضي:

- **اسم المستخدم:** `admin`
- **كلمة المرور:** `admin123`

> يُنصح بتغيير كلمة المرور فوراً من صفحة **الإعدادات ⚙️** بعد أول دخول.

---

## 🗂️ محتويات المشروع

| الملف / المجلد        | الوصف                                                        |
|------------------------|-------------------------------------------------------------|
| `config.php`           | إعدادات الاتصال بقاعدة البيانات (مضبوطة على XAMPP افتراضياً). |
| `database.sql`         | ملف إنشاء قاعدة البيانات والجداول والمستخدم الافتراضي.        |
| `index.php`            | الموجّه الرئيسي للصفحات.                                      |
| `login.php`            | صفحة تسجيل الدخول.                                            |
| `logout.php`           | تسجيل الخروج.                                                 |
| `includes/`            | ملفات مشتركة (الاتصال، الدوال المساعدة، الرأس، التذييل).       |
| `pages/`               | صفحات التطبيق (الرئيسية، الأصناف، الموردين، الأوردرات...).     |
| `ajax/`                | نقاط نهاية AJAX (إضافة صنف سريعاً).                           |
| `assets/`              | ملفات التنسيق (CSS) والجافاسكربت (JS).                        |

---

## ⚙️ تعديل إعدادات قاعدة البيانات (عند الحاجة فقط)

الإعدادات الافتراضية في `config.php` تناسب XAMPP مباشرةً:

```php
define('DB_HOST', 'localhost');
define('DB_NAME', 'purchase_tracker');
define('DB_USER', 'root');
define('DB_PASS', '');   // كلمة المرور فارغة في XAMPP
```

غيّرها فقط إذا كانت إعدادات MySQL لديك مختلفة.

## 🧭 أهم الشاشات

- **الرئيسية:** أزرار سريعة وإحصائيات وآخر الأوردرات.
- **أوردر جديد:** أهم شاشة — اختَر المورد والتاريخ وسعر الدولار، أضِف أصنافاً متعددة
  مع حساب الإجماليات مباشرةً، وأضِف صنفاً جديداً دون مغادرة الصفحة، ثم احفظ بضغطة واحدة.
- **بحث عن صنف:** سجل أسعار الصنف عبر كل الموردين مع تمييز أقل سعر ورسم بياني.
- **التقارير:** مقارنة أسعار صنف بين الموردين، والإنفاق حسب المورد/المجموعة/الشهر.
- **الإعدادات:** تغيير اسم المستخدم وكلمة المرور، بالإضافة إلى **النسخ الاحتياطي والاستعادة**.

---

## 💾 النسخ الاحتياطي والاستعادة

من صفحة **الإعدادات ⚙️**:

- **تنزيل نسخة احتياطية:** يُنشئ ملف `.sql` كاملاً يحوي كل بياناتك (الأصناف،
  الموردين، الأوردرات...) ويحفظه على جهازك. يُنصح بعمل نسخة بشكل دوري.
- **استعادة نسخة:** ارفع ملف `.sql` سبق تنزيله من التطبيق لاستعادة البيانات.

> ⚠️ الاستعادة **تحذف كل البيانات الحالية** وتستبدلها ببيانات الملف، لذا يظهر
> تأكيد قبل التنفيذ. بعد الاستعادة قد تحتاج لتسجيل الدخول مجدداً إذا تغيّرت بيانات الحساب.
>
> ملاحظة: إذا كان ملف النسخة كبيراً ولم يُقبل رفعه، ارفع قيمتي
> `upload_max_filesize` و`post_max_size` في `php.ini` ثم أعد تشغيل Apache.

## 🔒 ملاحظات الأمان

- كل كلمات المرور مُخزّنة مُشفّرة عبر `password_hash` (bcrypt).
- كل الاستعلامات تستخدم **Prepared Statements** للحماية من حقن SQL.
- كل النماذج محميّة برمز **CSRF**، وكل المخرجات مُهرّبة للحماية من **XSS**.
